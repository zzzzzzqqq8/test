# 简介

本固件适配斐讯 N1 旁路由模式，追求轻量（请注意：不具备 PPPoE、WiFi 相关功能）。<br>
固件包含默认皮肤、完整 IPv6 支持，以及下列 luci-app：<br>

- `luci-app-amlogic`
- `luci-app-argon-config`
- `luci-app-ddns-go`
- `luci-app-dockerman`
- `luci-app-https-dns-proxy`
- `luci-app-ksmbd`
- `luci-app-nft-qos`
- `luci-app-ttyd`
- `luci-app-uhttpd`
- `luci-ssl-openssl`
- `luci-theme-argon`
